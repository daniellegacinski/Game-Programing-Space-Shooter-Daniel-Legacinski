# 🚀 Space Shooter — Unity 2D Game

**Space Shooter** is a simple arcade-style space game created in **Unity**.  
The player controls a spaceship, flies through space, avoids enemies or obstacles, and tries to survive as long as possible.

This repository contains a ready Windows build, so the game can be launched directly without opening Unity Editor.

---

## 🎮 About the Project

This project was created as a small Unity game for learning and practice.  
The main idea is classic space shooter gameplay:

> Control the spaceship, avoid danger, shoot enemies, and survive.

The game is simple, but it demonstrates important Unity game development concepts such as movement, collisions, shooting mechanics, game objects, scenes, and building a playable `.exe` version.
